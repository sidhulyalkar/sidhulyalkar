# Hi there, I'm Sid! 👋

I'm a computational neuroscience engineer and developer interested in building tools that make complex biological data easier to process, understand, visualize, and reuse.

My background sits at the intersection of neuroscience, machine learning, cloud infrastructure, and data engineering. I have worked on large-scale multimodal neuroscience pipelines involving electrophysiology, calcium imaging, fiber photometry, pose estimation, behavior, and scientific data visualization.

I like building systems that turn messy, high-dimensional data into something interpretable, useful, and sometimes even beautiful.

Learn more about me at [sidhulyalkar.com](sidhulyalkar.com)

## 🧠 What I Work On
Neuroscience Data Systems: scalable pipelines for electrophysiology, calcium imaging, behavioral video, and multimodal experiments.
Machine Learning & Deep Learning: models for scientific data, neural signals, retrieval, and representation learning.
Knowledge Graphs & Search: tools for linking papers, datasets, concepts, and experimental provenance.
Data Visualization: interactive systems for exploring complex biological and behavioral data.
Cloud & Infrastructure: Docker, AWS, Kubernetes, databases, workflow orchestration, and reproducible scientific computing.

## 🚧 Currently Exploring
Neural search systems for discovering reusable neuroscience datasets and papers.
Knowledge graphs for mapping relationships between neuroscience concepts, experiments, and data.
Alternative tokenization methods for neural spike trains and foundation models.
Brain-computer interfaces, wearable sensing, EEG, and closed-loop neurofeedback.
Creative interfaces that blend neuroscience, music, light, and real-time visualization.
